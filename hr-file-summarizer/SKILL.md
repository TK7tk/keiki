# 人事ファイル要約スキル

## 概要

人事関連ドキュメントを500文字以内の日本語で要約する。対象ディレクトリ内のファイルを指定して、簡潔な要約を生成する。

## 対象ディレクトリ

```
/mnt/c/Users/西田貴彦/Documents/Cursor/西田個別作業/人事/02OUTPUT
```

## ワークフロー

1. ユーザーがファイル名を指定（部分一致可）
2. 対象ディレクトリ内でファイルを検索
3. ファイル内容を読み込み
4. 500文字以内の日本語で要約を生成

## 要約の形式

以下の形式で要約を出力する：

```
【タイトル】ファイル名

【要約】（500文字以内）
- 主題・テーマ
- 主要な論点・内容
- 結論・提言（あれば）
```

## 使用例

**ユーザー入力例:**
- 「孫子の兵法を要約して」
- 「哲学と経営のサマリー」
- 「人類はどこへ向かうのかを500文字で」

**実行手順:**
1. Globツールで対象ファイルを検索
2. Readツールでファイル内容を取得
3. 500文字以内で要約を作成して出力

## 注意事項

- 要約は必ず500文字以内に収める
- 日本語で出力
- ファイルが見つからない場合は、利用可能なファイル一覧を表示
